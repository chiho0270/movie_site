본 프로젝트는 대구가톨릭대학교의 서버 프로그래밍 과목 중 캡스톤디자인을 수행하는 팀 프로젝트입니다.

## 프로젝트 소개
### 팀 소개
+ 팀명: 네명이서하는마피아게임
+ 구성:
    + 이치호: 팀장/백
    + 신광호: 발표자료/프론트
    + 신주안: 자료조사/프론트
    + 이민혁: 자료조사/DB
    + 홍상은: 발표/백
### 개요
+ 과제명: 영화 평론사이트 제작 및 운영
+ 사이트명: 무비평
+ 목적: 영화에 대한 객관적인 평론 및 평점의 플랫폼 제작
+ 필요성: 코로나 팬데믹 이후 절감된 국내 영화 시장의 홍보 및 관객들의 객관적인 지표 제공
### 상세
+ 개발환경
  + Front-end: React
  + Back-end: FastAPI
  + DB: Amazon RDS (PostgreSQL)
  + API 연동: TMDB API
  + 배포 인프라: AWS
  + F-B: RESTful + axios
  + B-D: RESTful
